# go_robot
A full functioning robot that can move in all directions and also display the live camera transmission on the local network.

Hardware:
Raspberry Pi Model 2B V1.1
Raspberry Pi Camera
L298N Controller
Tamiya twin motors
Tamiya chasis

Software:
Python
HTML
CSS
Javascript
